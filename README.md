# Nike Sales Dashboard Analysis

## Project Overview
This project is an interactive sales dashboard created using Excel for analyzing Nike retail sales performance.
The dashboard provides insights into:
- Total Sales
- Units Sold
- Sales by Product
- Sales by State
- Sales Method Analysis
- Retailer Performance
- Regional Analysis

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design
- Data Cleaning

## Key Features
- Interactive filters using slicers
- Product-wise sales analysis
- State-wise performance tracking
- Sales value and units comparison
- Retailer performance insights
- Dynamic KPI cards

## Business Insights
- Men's Street Footwear generated the highest sales.
- Sales performance varied significantly across states.
- In-store sales contributed the largest share of total sales.
- Regional filters allow detailed sales analysis.
